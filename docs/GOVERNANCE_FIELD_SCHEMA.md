# Ops Observability Kit - 治理字段规范

最后更新：2026-02-27

## 目标
统一阻碍、依赖、门禁、评审记录的字段定义，确保多会话更新不跑偏。

## 阻碍字段规范（BLOCKER_REGISTER）
| 字段 | 必填 | 说明 |
|---|---|---|
| ID | 是 | `OB-O-XXX` |
| 级别 | 是 | S0/S1/S2/S3 |
| 类别 | 是 | 规范冲突/资源/合规/质量等 |
| 阻碍描述 | 是 | 具体且可复现 |
| 影响任务 | 是 | 任务 ID 列表 |
| 负责人 | 是 | 单一责任人 |
| 目标解除日期 | 是 | YYYY-MM-DD |
| 临时绕行方案 | 是 | 在解除前的可执行替代方案 |
| 状态 | 是 | OPEN/MITIGATING/RESOLVED/ACCEPTED_RISK |
| 证据链接 | 建议 | 文档路径或会话记录 |
| 复审日期 | 建议 | 下次复核日期 |

## 依赖字段规范（DEPENDENCY_MATRIX）
| 字段 | 必填 | 说明 |
|---|---|---|
| 依赖ID | 是 | `DEP-O-XXX` |
| 依赖项 | 是 | 可验证对象 |
| 类型 | 是 | 环境/配置/数据/组织/集成 |
| 影响任务 | 是 | 任务 ID 列表 |
| 验证方式 | 是 | 命令/检查动作 |
| 当前状态 | 是 | READY/PENDING/BLOCKED |
| 负责人 | 是 | 单一责任人 |
| 目标就绪日期 | 是 | YYYY-MM-DD |
| 证据链接 | 建议 | 验证记录位置 |
| 备注 | 否 | 风险或上下文 |

## 评审字段规范（GO_NO_GO_REVIEW）
| 字段 | 必填 | 说明 |
|---|---|---|
| 评审日期 | 是 | YYYY-MM-DD |
| 结论 | 是 | Go/No-Go |
| 触发条件 | 是 | 直接导致结论的事实 |
| 解锁条件 | 是 | 下一次转 Go 的最小条件 |
| 下次复评日期 | 是 | YYYY-MM-DD |
| 评审人 | 是 | 责任人 |

## 变更规则
1. 新增字段必须同步更新本规范。
2. 任何台账字段缺失，不得判定为 Go。
3. 证据链接缺失可临时允许，但需在下次会话补齐。
