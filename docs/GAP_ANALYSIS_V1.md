# Ops Observability Kit - 差距与风险报告 V1

最后更新：2026-03-02

## 差距清单
| 编号 | 差距项 | 影响 | 优先级 | 建议动作 |
|---|---|---|---|---|
| GAP-001 | 本地二进制尚未安装，运行证据不足 | 无法完成全链路实测 | 高 | 按需安装或接入现有运行环境 |
| GAP-002 | 告警通道未联通 | 无法验证通知时延 | 中 | 明确邮件/IM 接收器并演练 |
| GAP-003 | 真实业务流量样本不足 | 阈值难以精准定标 | 中 | 先用试点服务基线数据回填 |
| GAP-004 | 部分演练报告仍为模板 | 缺少可审计复盘证据 | 中 | 执行 M1/M2/M3 演练并填报 |

## 风险摘要
1. 若长期仅离线产出，后续联调集中爆雷概率上升。
2. 若不尽早收集真实基线，告警阈值将长期偏保守或偏激进。
